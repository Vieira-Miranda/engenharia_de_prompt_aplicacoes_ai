# engenharia_de_prompt_aplicacoes_ai
Atividades da Faculdade

Bem‑vindo ao repositório oficial do Curso de Engenharia de Prompt e Aplicação em IA, criado para ajudar profissionais, estudantes e entusiastas a dominarem o uso estratégico de IA Generativa em cenários reais.
Este repositório reúne materiais, exercícios, frameworks, prompts avançados e projetos práticos utilizados durante o curso.
